# BAAB

Baab står for barneanestesi og akuttbehandling er et elektronisk støtteverktøy eller "app" utviklet for bruk på mobil, nettbrett eller PC. Den blir som en lokal utgave av Cope-appen med litt ekstra funksjonalitet : )

Appen er fortsatt under utvikling og hverken kode eller parametere er gjennomgått av andre enn utvikler. Selv når appen er ferdigutviklet vil den kun være et hjelpemiddel. Oppgitte verdier er veiledende, og feil kan forekomme.

Appen er basert på <a href="https://github.com/anaestheticsapp/calculator">åpen kildekode</a> av <a href="https://anaesthetics.app/">AnaestheticsApp</a>, under <a href="https://www.isc.org/licenses/">ISC-lisens</a>

Hvis du ønsker å bidra med innspill, feilrettinger, forslag til ny funksjonalitet eller har andre kommentarer er det bare å sende en <a href="mailto:post@863.no">mail</a>
